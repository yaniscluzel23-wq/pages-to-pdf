# pages-to-pdf

Petit script Python pour convertir un fichier `.pages` en `.pdf` sur macOS en utilisant Apple Pages.
Pour vous evitez de payer apres 3 conversion
EN esperant que cela vous plaise 

## Prérequis

- macOS
- Apple Pages installé
- Python 3

## Utilisation

python3 pages_to_pdf.py /path/to/file.pages
ou 
python3 pages_to_pdf.py "~/Desktop/file.pages"
ou une troisieme solution : 
python3 pages_to_pdf.py  puis glissez le fichier .pages dans le terminal
